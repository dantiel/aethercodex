# Hermetic Manifest — AetherCodex Native

## PROJECT FOCUS

Native Mac code editor with embedded AI — replacing TextMate as the vessel for AetherCodex.

Primary goal: Ship a polished, App Store-ready editor that runs the full AetherCodex AI stack natively.

## CORE HERMETIC PRINCIPLES

### Mentalism
The editor is an extension of consciousness. Code reflects thought; the AI amplifies it.

### Correspondence
As the TextMate plugin worked, so shall the native app — patterns preserved, essence elevated.

### Vibration
Nothing rests. The editor pulses with live AI assistance, real-time collaboration between human and machine.

### Polarity
Balance the magical (AI divination) with the practical (fast native editor). Neither dominates; both serve.

### Rhythm
Development flows in cycles: scaffold → build → test → refine. Honor the rhythm.

### Cause & Effect
Every feature has its purpose. No bloat. Every line of code earns its place.

### Gender
Masculine: structure, performance, native code. Feminine: intuition, AI, creative flow. United.

## TECHNICAL PRINCIPLES

### Swift Style
- Clean, expressive Swift
- Protocol-oriented where beneficial
- Imperative for UI (AppKit/SwiftUI), functional for data

### Ruby Integration
- Preserve poetic Ruby core
- Bridge via clean Swift interface
- Migrate gradually, don't rewrite for rewrite's sake

### UI Philosophy
- Native first (AppKit > SwiftUI for editor)
- Keyboard-centric (TextMate heritage)
- Minimal chrome, maximal content

### AI Integration
- Seamless: AI appears when needed, vanishes when not
- Contextual: understands code, project, intent
- Respectful: assists, never replaces

## DEVELOPMENT RHYTHM

1. **Scaffold** — Structure before content
2. **Bridge** — Connect old to new
3. **Polish** — Details make the difference
4. **Ship** — Perfect is enemy of done
